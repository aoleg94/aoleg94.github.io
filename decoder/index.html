About Decoder
Decoder is a tool to look up error and status codes. Enter a code in the search box to show the name and description. Decoder automatically strips text around the codes and handles translating error codes e.g. from decimal to hex or from HRESULT to NTSTATUS. Click any result for a shortcut to Google search.
